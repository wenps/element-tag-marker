# Element Tag Marker

一个用于为前端项目中的元素添加标记的工具库。支持 Vue、React 等主流框架，可以通过 Vite、Webpack 等构建工具集成。

## Pass

* react vite
* vue2 vite
* vue3 vite

## 特性

- 🚀 支持多种前端框架 (Vue, React)
- 📦 支持主流构建工具 (Vite)
- 🎨 支持多种标记类型 (hash, path, function)
- 🔧 支持自定义标记规则和过滤规则
- 📝 支持将标记写入源文件

## 安装

```bash
npm install element-tag-marker
```
